这是一个可以对node.js服务器进行简易操作的简单的桌面应用（实际上只是一个小demo）

应用名：nodeserver
开发语言：Qt/MinGW64
运行环境：Windows 10
版本：1.0

源代码在resouce文件夹内，项目文件在目录resource/node-set内，qmake的构建内容在resource/build-nodeserver-Desktop_Qt_5_12_1_MinGW_64_bit-Release内
